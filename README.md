# Usage
Just some configurations in my PC. Included:  
* VIM
* TMUX
* VSCODE 

# Install
```
git clone git@github.com:StepfenShawn/Configuration.git
```
And then:
```sh
cd Configuration
cp .vimrc ~/.vimrc
cp .tmux.conf ~/.tmux.conf
cp .bashrc ~/.bashrc
cp .minttyrc ~/.minttyrc
```
Or:  
```sh
cd Configuration
./install.sh
```
